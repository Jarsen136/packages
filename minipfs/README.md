## Usage

Install package:

```sh
# npm
npm install @kodadot1/minipfs

# yarn
yarn install @kodadot1/minipfs

# pnpm
pnpm install @kodadot1/minipfs
```

Import:

```js
// ESM
import { } from '@kodadot1/minipfs'

// CommonJS
const { } = require('@kodadot1/minipfs')
```
